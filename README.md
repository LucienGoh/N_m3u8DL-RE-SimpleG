# N_m3u8DL-RE-SimpleG
N_m3u8DL-RE's simple GUI

20230125：

本人不是软件工程师，也没有界面相关的编程经验，纯粹是有点好奇所以就从原软件库中拉取过来工程把相关参数做了修改，建立这个工程主要是为了简单测试下github的发布功能。

软件使用方式和原来的 simple GUI 一模一样，只是把原来工程使用的 N_m3u8DL-CLI.exe 更换为 N_m3u8DL-RE.exe，所以不做更多说明了。
